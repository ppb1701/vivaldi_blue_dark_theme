# Blue Dark Theme for Vivaldi

A beautiful dark theme for Vivaldi browser featuring a blue tint and rounded tabs for a modern, sleek appearance.

![Theme Preview](https://img.shields.io/badge/Vivaldi-Theme-0099ff?style=for-the-badge)

## Theme Features

- **Dark blue color scheme** - Easy on the eyes with a sophisticated blue tint
- **Rounded tabs** - Modern, polished look with 14px border radius
- **Optimized contrast** - Carefully selected colors for readability
- **Accent colors** - Vibrant blue highlights (#0099ff) for active elements

### Color Palette

- Background: `#051629`
- Accent Background: `#001229`
- Window Background: `#255372`
- Foreground Text: `#a5bff3`
- Highlight: `#0099ff`

## Installation Instructions

### Method 1: Install from Vivaldi Theme Store (Easiest)

1. **Visit the theme page**
   - Go to [Blue Dark Theme on Vivaldi Themes](https://themes.vivaldi.net/themes/PBylqEWDlr5)

2. **Install with one click**
   - Click the `Install Theme` button
   - The theme will be automatically downloaded and applied to your browser

### Method 2: Import Theme File

1. **Download the theme file**
   - Go to the [latest release](https://github.com/ppb1701/vivaldi_blue_dark_theme/releases/tag/theme)
   - Download `settings.json`

2. **Open Vivaldi Settings**
   - Click on the Vivaldi menu (top-left corner)
   - Navigate to `Tools` â†’ `Settings` (or press `Ctrl+F12` / `Cmd+,` on Mac)

3. **Go to Themes**
   - In the Settings sidebar, click on `Themes`

4. **Import the theme**
   - Scroll down and click the `+` button next to "My Themes"
   - Select `Import Theme`
   - Choose the downloaded `settings.json` file
   - The theme will be imported and automatically applied

### Method 3: Manual Configuration

If you prefer to create the theme manually:

1. **Open Vivaldi Settings** (`Ctrl+F12` / `Cmd+,`)
2. **Go to Themes** section
3. **Create a new theme** by clicking the `+` button
4. **Configure the following settings**:
   - **Name**: Blue Dark Theme
   - **Background**: `#051629`
   - **Foreground**: `#a5bff3`
   - **Highlight**: `#0099ff`
   - **Accent**: `#001229`
   - **Corner Rounding**: 14
   - Enable "Use Accent Color on Window"

5. **Save** your theme

## Troubleshooting

**Theme doesn't apply after import:**

- Ensure you're using Vivaldi version 3.0 or higher
- Try restarting the browser after importing

**Colors look different:**

- Check your system display settings and color profile
- Ensure hardware acceleration is enabled in Vivaldi settings
